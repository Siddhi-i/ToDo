# To-Do List Web App

This is a front-end To-Do List application built using **HTML, CSS, and JavaScript**.
The app allows users to add tasks with priority levels, mark them as completed, filter tasks, and clear completed items. All updates happen dynamically without reloading the page.

## How to Run the Project

1. Open the project folder in VS Code
2. Right-click on `index.html`
3. Select **Open with Live Server**
4. The application will open in the browser

---

## How It Works

* User enters a task and selects a priority level
* Clicking **Add** creates a task dynamically in the list
* Clicking a task marks it as completed
* Filter buttons show all, active, or completed tasks
* Task counter updates automatically
* **Clear Completed** removes all completed tasks at once
* JavaScript handles all DOM updates without page refresh
